## 故障排除

### 常见问题

1. **HyperLPR3初始化失败**
   - 检查网络连接，首次运行需要下载模型
   - 尝试重新运行，可能需要多次尝试

2. **PaddleOCR初始化失败**
   - 检查 paddlepaddle 和 paddleocr 版本兼容性
   - 尝试使用CPU版本: `pip install paddlepaddle`

3. **识别精度不佳**
   - 检查图片质量和车牌清晰度
   - 调整置信度阈值
   - 尝试不同的识别引擎

## Web 管理后台

系统提供 Web 管理界面，用于查看统计、违停记录、配置禁停区域和系统参数。

### 启动 Web 后台（未完善）

```bash
# 在项目根目录 (plate_detection) 下运行
python web/run_web.py
```

访问 http://127.0.0.1:5000 打开管理界面。

在终端运行 configure_zones.py 你的文件路径以画出违停区域
在终端运行 parking_violation_system.py 你的文件路径以运行

### 功能说明

- **仪表盘**: 查看处理帧数、检测车辆、识别车牌、违停次数
- **任务控制**: 启动/停止检测任务，指定视频或图片文件夹路径
- **违停记录**: 查看历史违停记录列表
- **禁停区域**: 查看禁停区域配置（可用 configure_zones.py 配置）
- **系统配置**: 修改违停判定阈值、帧跳过等参数
